# CHIDA Administrative Management System (CDAMS)

## 项目简介
CDAMS（CHIDA行政管理系统）是一个综合性的企业行政管理平台，旨在提供高效、便捷的行政事务管理解决方案。

## 主要功能模块

### 1. 车辆管理系统
- 车辆信息管理（新增、编辑、删除）
- 车辆通行证批量管理
- Excel导入导出功能
- 车辆状态追踪（待审批、已审批、已驳回等）
- PDF文档生成
- 批量删除功能
- 分页查询与筛选

### 2. 宿舍管理系统
- 宿舍楼管理
- 房间信息管理
- 住户信息管理
  - 入住登记
  - 退住手续
  - 住户信息编辑
- 宿舍长管理
- 历史住户记录追踪
- 房间使用状态监控
